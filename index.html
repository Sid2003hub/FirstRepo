2. Task Tracker Web App

The Task Tracker Web App is a simple productivity tool developed using HTML, CSS, and JavaScript. It allows users to create, edit, delete, and mark tasks as complete. All tasks are stored in the browser’s localStorage, so they remain saved even after refreshing the page. The app helps users manage their daily tasks efficiently and is built with a user-friendly interface. This project introduces concepts like DOM manipulation, event handling, and data persistence in web development.
